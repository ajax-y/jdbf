> A next-generation, real-time ecosystem designed to scale campus communities. Club Hub transforms manual club management into a high-performance, automated "Node" system.

## 🚀 Vision
Club Hub isn't just a dashboard; it's a **Living Community Engine**. It bridges the gap between physical campus events and digital participation using high-fidelity real-time synchronization.

## ✨ Core Features (Judge's Guide)

### 1. 🟢 The Live Node Scanner (Events)
*   **Encrypted Ingress:** Secure QR-based check-in system.
*   **Merit Point Economy:** Real-time awarding of "Merit Points" (10 per event) that dynamically update user ranks.
*   **Visual Celebration:** Integrated celebratory feedback upon successful synchronization.
*   **Cloud Archives:** Instant access to personal check-in history.

### 2. 🏛️ The Innovation Gallery (Projects)
*   **Decentralized Repo:** A centralized home for every club project.
*   **Social Interaction:** Like, share, and comment on peer prototypes in real-time.
*   **Tech Stack Filter:** High-speed curation with smart tagging (AI/ML, Blockchain, Web3, etc.).

### 3. 📊 Real-Time Hall of Fame (Leaderboard)
*   **Dynamic Tiers:** Automatic progression through ranks (Bronze → Silver → Gold → Diamond).
*   **Global Activity Feed:** A live scroll of every "Handshake" and "Deployment" happening across the club.

### 4. 📺 The Live Wall (Projector View)
*   **Ingress Stream:** A high-performance broadcast screen for event organizers. 
*   **Live Participation Index:** Tabular counts and participant cards update instantly on the wall as members scan the QR.

### 5. 🛠️ Administrative Command Center
*   **Node Initialization:** Create events with secure secrets and custom branding.
*   **Global Signals:** One-click notifications that hit every member's hub instantly.
*   **Security Management:** Toggle event nodes on/off (Closed Node logic) for precise control.

## 🚦 Installation & Deployment

1. **Clone the Hub:** `git clone https://github.com/ajax-y/jdbf.git`
2. **Environment Sync:** Setup `.env.local` with your Supabase credentials.
3. **Database Handshake:** Apply `supabase_setup.sql` to initialize the cloud architecture.
4. **Initiate Core:** `npm run dev` (or `bun/yarn/pnpm`)
